# 便签应用 Demo Brief

## Goal

A mobile-style notes app. Single user, mock data only.

## Pages

1. **列表 / Notes**: card grid of notes with title, snippet, updated time; search box filters by title.
2. **编辑 / Editor**: title + body textarea, save returns to list with the note updated (in-memory).

## Style

- Mobile viewport (390x844), minimal, ink-blue accent, Chinese UI text.

## Constraints

- Seed with 8 notes. No backend, no persistence beyond page session.

## Acceptance

- Search box visibly filters the note list.
- Editing a note updates its card on the list page.
